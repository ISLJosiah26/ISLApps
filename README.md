# PostIQ (React Apple-Glass Analyzer)

This project now runs as a React app (loaded via CDN in `index.html`) with an Apple-style glass UI for multi-post social performance analysis.

## Included in this version

- React-based multi-post workflow.
- Platform-aware post formats (LinkedIn, Instagram, Facebook).
- Glassmorphism design system and Apple-style controls.
- Drag/drop + click image upload with preview per post.
- Engagement rate visualization bar.
- Anthropic API call flow for structured AI analysis and ranked output.

## Run

Open `index.html` in a browser with internet access (for React/Babel/Google Fonts CDNs).
